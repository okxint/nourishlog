# NourishLog

Your food diary that actually gets Indian food. Track what you eat by chat, photo, or text - 500+ Indian dishes with IFCT nutrition data.

**Live:** [nourishlog.vercel.app](https://nourishlog.vercel.app)

## Features

- **Chat-based logging** - type "had biryani for lunch" and get full nutrition breakdown
- **Photo upload** - snap your plate, we identify and log it
- **500+ Indian dishes** - masala dosa to chole bhature, with IFCT-sourced nutrition data
- **Query your history** - "what did I eat yesterday?", "when did I last have biryani?"
- **Edit any value** - calories, protein, carbs, fat, fiber - all editable inline
- **Custom recipes** - add your own dishes with exact macros
- **Dashboard with filters** - date range, meal type, food search, clickable charts
- **CSV export** - download your data as a spreadsheet
- **Daily challenges & achievements** - streaks, badges, share progress
- **Water tracking** - track glasses on the home page
- **Dark & light mode** - toggle anytime

## Tech Stack

- Next.js 16 (App Router)
- TypeScript
- Tailwind CSS
- Recharts
- localStorage (no backend needed)

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).
